# Envelope CAD assets

STEP files in this directory are researched envelope assets for the Layout and Envelope tabs. The 20 ft Conex asset is an extracted STEP payload from the free cad-blocks-studio-jp `20ft_container-STEP` ZIP package. The 40 ft and high-cube Conex options also reference public CAD libraries discovered during the asset search, including Marathon OS STEP metadata and the FreeCAD-library 20 ft ISO container assembly; structures without redistributable STEP downloads use app-owned parametric STEP proxies with source traceability in the file header.
